=== Student Roster Management System

A C++ application designed to manage student records for a university course roster. This project demonstrates object-oriented programming principles, data parsing, and data management using custom classes and arrays.

## Features

* Store and manage student information
* Add new students to the roster
* Remove students by ID
* Validate student email addresses
* Calculate average days spent in courses
* Filter students by degree program
* Display formatted student records

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Visual Studio
* Arrays and Pointers

## Project Structure

### Student Class

Represents an individual student and stores:

* Student ID
* First Name
* Last Name
* Email Address
* Age
* Days to Complete Courses
* Degree Program

### Roster Class

Manages the collection of students and provides functionality to:

* Parse student data
* Add students
* Remove students
* Print all student records
* Validate emails
* Calculate average course completion times
* Filter students by degree type
